

适配器模式: Adapter

    生活中我们也经常用到适配器，比如我们买的手机为了便宜一些会去买港行，但是港行的缺点就是那边的插头是三角插头，与咱们这边的规格不同。

    这时候就需要拿转换头转换才可以用，这个转换过程就是适配器的作用。

    下面我们会以上为例，如何把三角插头(香港规格)转换成国内二脚插头（比喻），来理解适配器模式。

    三角插头：ThreePlug
    二脚插头：TwoPlug
    适配器：PlugAdapter


    适配器模式分为：适配器（继承）模式   和   适配器（代理）模式

    适配器（继承）模式: adapterextends

    适配器（代理）模式: adapterdepute
